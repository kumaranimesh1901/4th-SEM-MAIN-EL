ℹ️ System Architecture

Smart Firewall & IDS Architecture

This system uses a hybrid approach for threat detection:


🔹 Rule-Based Engine
Pattern matching for DDoS, Port Scan, SYN Flood, SQL Injection, Brute Force, DNS Tunneling.


🔹 Isolation Forest (Anomaly)
Unsupervised anomaly detection on flow features. Learns normal behavior and flags deviations.


🔹 XGBoost Classifier
Classifies traffic as Normal, DDoS, Ransomware, APT, SQL Injection, or Brute Force.


🔹 Decision Engine
Rule + ML both confirm → BLOCK (auto-firewall)
Single engine detection → FLAG (manual action)
Repeated FLAGs (×3) → Auto-BLOCK
Neither → ALLOW


🔹 Telegram Alerts
Sends real-time notifications for HIGH/CRITICAL severity detections to Telegram.
